  3  """ vm_utilization.py:  Captures utilization data for virtual machines & hosts;   
  4  and stores the information in a RRD database.  
  5  RRD is a Round Robin Database which is created for pre-defined time interval.  
  6  When new data reaches the starting point, it overwrites existing data. The RRDtool

 11

 13  Memory 
 14  CPU

 16  Disk (read & write) 
 17
